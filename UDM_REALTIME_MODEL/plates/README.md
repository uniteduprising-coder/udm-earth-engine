# UDM Master Plates (immutable geometry)

Copy the NASA bathymetric UDM plates here:

- `1000083851.jpg` — full-disk master coordinate plate (1536×1533)
- `1000083849.jpg` — central zoom / north-axis aperture calibration (1151×1536)

These files are **not** generated by the pipeline. The plate lock prevents any external source from replacing them.

Until both files are present, `scripts/run_realtime_model.py` runs in scaffold mode using `config/udm_master_plate.yaml` initial constants only.
